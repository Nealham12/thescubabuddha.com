# The Scuba Buddha — Production Website v1 ## Upload to GitHub 1. Extract this ZIP. 2. Open your GitHub repository. 3. Click **Add file → Upload files**. 4. Drag every file and folder from inside this extracted folder into GitHub. 5. Commit the changes. 6. In **Settings → Pages**, deploy from the `main` branch and `/ (root)`. ## Before final launch - Replace `images/hero-placeholder.svg` with the selected underwater hero photo. - Replace `images/owner-placeholder.svg` with Neal's portrait. - Add the final Google Workspace email address to `contact.html`. - Replace the Formspree placeholder in `contact.html`. - Add final prices and turnaround times to `services.html`. - Verify the Facebook URL. - Review course names, certification availability, and service wording. - Do not delete the `CNAME` file when using thescubabuddha.com. ## Important DNS note When pointing the Wix-registered domain to GitHub Pages, preserve all Google Workspace MX and verification records.
